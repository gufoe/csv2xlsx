# Csv to Excel conversion - easy and fast
```sh
USAGE:
    csv2xlsx [OPTIONS] --input <INPUT>

OPTIONS:
    -c, --column-size <COLUMN_SIZE>    by default it depends on the content size - only useful if
                                       output is an excel file
    -h, --help                         Print help information
    -i, --input <INPUT>                
    -n, --sheet-name <SHEET_NAME>      sheet name to write (or read if converting to csv)
    -o, --output <OUTPUT>              [default: output.xlsx]
    -s, --sheet <SHEET>                index of the sheet to be read if converting to csv [default:
                                       0]
    -t, --to-csv                       convert <input> to a csv file
    -V, --version                      Print version information

```
